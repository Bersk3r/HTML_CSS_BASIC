# HTML은 트리 구조로 되어 있음 (부모-자식-형제, 계층 구조)
# 태그 구조는 하기와 같습니다.
<parent>
    <children>
        <grandchildren>
        <grandchildren>
    </children>
    <sibling>
        <grandchildren>
        <grandchildren>
    </sibling>
</parent>

# HTML 내 여는 태그만으로 동작하는 경우, 내부에 들어갈 태그나 내용이 없을 때에 해당됨
=> <meta charset="utf-8" />
=> 여는 태그만 작성해도 동작하더라도 닫는 태그는 반드시 작성할 것
